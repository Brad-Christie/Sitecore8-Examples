CORS Example

Because this page runs from another domain (it's probably loaded from file:/// protocol) we can showcase the CORS (Cross-Origin Resource Sharing) features. If you see an error when executing these examples, one of two things went wrong: CORS isn't enable, or you never pushed the code to your server.

Please remember that th einstance I've used is sc8r141212, so if yours varies you'll need to adjust it in the options drop-down at the top-right of the page.
